Aplicación web Animal-Cosver

Este proyecto consistió en el desarrollo y diseño de una aplicación web de un sistema experto. El objetivo era crear una aplicación web que mencione qué enfermedad tiene una máscota en base a los sintomas que el cliente haya seleccionado, y que el cliente pueda crear una cuenta, iniciar sesión y guardar el tratamiento que el sistema experto le haya mostrado. Utilizamos tecnologías modernas para garantizar una experiencia de usuario óptima y un diseño adaptativo que se ve bien en todos los dispositivos.
